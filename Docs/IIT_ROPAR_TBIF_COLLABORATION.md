## 10. Internal Notes

### IP Sharing at IITs (General Norms)

- IP created before incubation: Founder owns 100%
- IP created using IIT facilities: Startup owns, IIT gets equity (2-5%) as consideration
- IP from faculty-supervised research: IIT owns, grants exclusive license to startup
- IP from student BTP/MTP: IIT technically owns, equity stake covers it
- Sponsored research (startup pays): Startup gets first right to license, IIT takes 10-20% overhead

**Our approach:** Negotiate upfront that all IP developed during incubation is startup-owned, with equity covering facility usage. For faculty co-developed IP, establish clear co-ownership terms before work begins.
