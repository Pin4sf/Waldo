## Tier 1: Direct Competitors

### Nori (YC) — Closest Threat

"Peter Attia in your pocket." Serial founders with 2x exits (Chartable → Spotify). **Live on App Store.**

**What they do better:** Already shipped. Multi-source data (nutrition, blood tests, medical records). Daily plan format.

**Where we're better:** Proactive channel delivery (Telegram/WhatsApp). CRS grounded in SAFTE-FAST. Agent memory that evolves. Real-time stress detection. MCP server path.

**Steal:** Daily plan format for Morning Wag. Weekly review as Phase D feature.

### Prana Health (YC W26) — "Always-On AI Doctor"

Detects "clinical drift" (subtle health decline over weeks). 800+ healthcare institutions via Fasten. Physician escalation network.

**Where we're better:** Consumer UX (not clinical). Channel delivery. Non-medical positioning.

**Steal:** "Clinical drift" language for Constellation. Fasten for EHR integration.

### Galen AI (YC) — Clinical + Wearable Agent

800+ medical institutions, 20+ wearables, Dexcom/CGM integration.

**Steal:** CGM/Dexcom integration (Phase 2). Personal health timeline concept.

## What to Steal (Top 5)

1. **Nori's daily plan format** — Morning Wag should be a structured plan, not just an insight
2. **Prana's "clinical drift" language** — "Your recovery has been drifting down for 12 days" is more powerful than "pattern detected"
3. **PHIA's code generation** — Dynamic Worker for numerical precision (proven 10% accuracy gain)
4. **Open Wearables MCP patterns** — Reference for "Waldo as MCP server" design
5. **CUDIS gamification** — Health points, streaks, achievements boost retention
